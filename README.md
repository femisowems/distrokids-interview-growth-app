# DistroKid Growth OS

A cinematic, experiment-driven marketing system for music launches.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Supabase
- PostHog
- OpenAI API
- Zustand
- React Hook Form
- Zod

## What’s included

- Immersive launch homepage
- Campaign studio with AI brief generation
- Landing page renderer and dynamic release route
- Funnel, analytics, and attribution views
- Experiment dashboards
- SEO utilities and sitemap generation
- Internal CMS surface
- Supabase schema
- Analytics and AI utility layers

## Environment

Copy `.env.example` to `.env.local` and fill in your keys.

## Run

```bash
npm install
npm run dev
```
